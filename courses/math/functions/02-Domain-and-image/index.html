Domínio e Imagem de Funções

Em matemática, o domínio e a imagem são conceitos fundamentais para entender completamente qualquer função. Eles definem, respectivamente:

  • Domínio (D(f)): Todos os valores de entrada (x) que a função pode aceitar
  • Imagem (Im(f)): Todos os valores de saída (f(x)) que a função pode produzir

Domínio de uma Função

O domínio é determinado pelas restrições da função:

  • f(x) = \frac{1}{x} → D(f) = ℝ - {0} (não pode dividir por zero)
  • f(x) = \sqrt{x} → D(f) = [0, +∞[ (raiz quadrada exige x ≥ 0)
  • f(x) = \log(x) → D(f) = ]0, +∞[ (logaritmo exige x > 0)

Imagem de uma Função

A imagem é determinada pelos valores que a função pode atingir:

  • f(x) = x^2 → Im(f) = [0, +∞[ (quadrados são sempre positivos)
  • f(x) = \sin(x) → Im(f) = [-1, 1] (seno varia entre -1 e 1)
  • f(x) = e^x → Im(f) = ]0, +∞[ (exponencial é sempre positiva)

Representação Gráfica

No gráfico de uma função:

  • Domínio: Projeção do gráfico no eixo x (horizontal)
  • Imagem: Projeção do gráfico no eixo y (vertical)

Como Determinar Domínio e Imagem

Para o Domínio:

  1. Identifique restrições na função
  2. Verifique denominadores (não podem ser zero)
  3. Analise expressões sob raízes pares (devem ser ≥ 0)
  4. Verifique argumentos de logaritmos (devem ser > 0)

Para a Imagem:

  1. Analise o comportamento da função
  2. Identifique valores mínimos e máximos
  3. Observe assíntotas horizontais
  4. Estude transformações gráficas

Exemplo Completo

Considere a função f(x) = \sqrt{4 - x^2}

Domínio:
4 - x² ≥ 0 ⇒ x² ≤ 4 ⇒ -2 ≤ x ≤ 2
D(f) = [-2, 2]

Imagem:
Como √(4 - x²) varia de √0 a √4 ⇒ 0 ≤ f(x) ≤ 2
Im(f) = [0, 2]
