Exercícios de Domínio e Imagem

1. Domínio Básico Fácil

Determine o domínio das seguintes funções:

  1. f(x) = 3x + 2
  2. g(x) = \frac{1}{x-4}
  3. h(x) = \sqrt{x+5}

Respostas:

a) D(f) = \mathbb{R} (Todos os números reais)
b) D(g) = \mathbb{R} - \{4\} (Todos os reais exceto x=4)
c) D(h) = \{x \in \mathbb{R} \mid x \geq -5\} (x+5 ≥ 0)

2. Imagem Básica Fácil

Determine a imagem das seguintes funções:

  1. f(x) = x^2 + 1
  2. g(x) = -2x + 3
  3. h(x) = \sqrt{9 - x^2}

Respostas:

a) Im(f) = [1, +\infty[ (x² ≥ 0 ⇒ x²+1 ≥ 1)
b) Im(g) = \mathbb{R} (Função linear)
c) Im(h) = [0, 3] (√(9-x²) varia de 0 a 3)

3. Domínio com Múltiplas Restrições Médio

Determine o domínio da função:

f(x) = \frac{\sqrt{x-2}}{x^2 - 9}

Dica: Considere tanto a raiz quadrada quanto o denominador.

Resposta:

Duas condições devem ser satisfeitas simultaneamente:

  1. Expressão sob a raiz: x - 2 \geq 0 \Rightarrow x \geq 2
  2. Denominador diferente de zero: x^2 - 9 \neq 0 \Rightarrow x \neq \pm 3

Portanto: D(f) = [2, +\infty[ - \{3\} = [2, 3[ \cup ]3, +\infty[

5. Função por Partes Difícil

Dada a função:

f(x) = \begin{cases} x + 1 & \text{se } x < 0 \\ x^2 & \text{se } 0 \leq x \leq 2 \\ 4 & \text{se } x > 2 \end{cases}

Determine:

  1. O domínio
  2. A imagem

Respostas:

a) D(f) = \mathbb{R} (definida para todos os reais)

b) Analisando cada parte:

  • Para x < 0: f(x) = x + 1 \Rightarrow Im = ]-\infty, 1[
  • Para 0 ≤ x ≤ 2: f(x) = x^2 \Rightarrow Im = [0, 4]
  • Para x > 2: f(x) = 4 \Rightarrow Im = \{4\}

União das imagens: Im(f) = ]-\infty, 1[ \cup [0, 4] = ]-\infty, 4]

6. Função Racional Difícil

Determine domínio e imagem da função:

f(x) = \frac{2}{x - 1} + 3

Respostas:

a) Domínio: D(f) = \mathbb{R} - \{1\} (denominador não pode ser zero)

b) Imagem: Vamos analisar a função:

y = \frac{2}{x-1} + 3

Isolando x:

y - 3 = \frac{2}{x-1}

x - 1 = \frac{2}{y-3}

x = \frac{2}{y-3} + 1

O denominador não pode ser zero: y - 3 \neq 0 \Rightarrow y \neq 3

Portanto: Im(f) = \mathbb{R} - \{3\}
