Exercícios sobre Domínio e Imagem

1. Identificação de Domínio Fácil

Determine o domínio das seguintes funções:

  1. f(x) = 3x - 7
  2. g(x) = \frac{1}{x+2}
  3. h(x) = \sqrt{x-4}

Respostas:

a) D(f) = \mathbb{R} (Todos os números reais)

b) D(g) = \mathbb{R} \setminus \{-2\} (Todos os reais exceto x = -2)

c) D(h) = [4, +\infty[ (x ≥ 4)

2. Determinação de Imagem Médio

Encontre a imagem das funções:

  1. f(x) = x^2 + 3
  2. g(x) = -\sqrt{x}
  3. h(x) = \frac{2}{x-1}
Dica: Para a letra c, observe que o denominador nunca será zero no domínio.

Respostas:

a) Im(f) = [3, +\infty[ (x² ≥ 0 ⇒ x²+3 ≥ 3)

b) Im(g) = ]-\infty, 0] (√x ≥ 0 ⇒ -√x ≤ 0)

c) Im(h) = \mathbb{R} \setminus \{0\} (2/(x-1) nunca será zero)

3. Domínio e Imagem Combinados Médio

Para a função f(x) = \sqrt{9 - x^2}, determine:

  1. O domínio
  2. A imagem

Respostas:

a) D(f) = [-3, 3] (9 - x² ≥ 0 ⇒ x² ≤ 9 ⇒ -3 ≤ x ≤ 3)

b) Im(f) = [0, 3] (√(9-x²) varia de √0=0 a √9=3)

4. Função Racional Difícil

Considere a função f(x) = \frac{x+1}{x-2}:

  1. Determine seu domínio
  2. Mostre que sua imagem é \mathbb{R} \setminus \{1\}
Dica: Para a letra b, tente isolar x em função de y.

Respostas:

a) D(f) = \mathbb{R} \setminus \{2\} (x-2 ≠ 0 ⇒ x ≠ 2)

b) Seja y = \frac{x+1}{x-2}. Resolvendo para x: y(x-2) = x+1 ⇒ yx - 2y = x + 1 ⇒ yx - x = 2y + 1 ⇒ x(y-1) = 2y+1 ⇒ x = \frac{2y+1}{y-1} O denominador y-1 não pode ser zero ⇒ y ≠ 1. Logo, Im(f) = \mathbb{R} \setminus \{1\}

5. Função por Partes Difícil

Dada a função definida por partes:

f(x) = \begin{cases} x^2 & \text{se } x \leq 1 \\ 2x - 1 & \text{se } x > 1 \end{cases}

  1. Determine o domínio
  2. Encontre a imagem
  3. Esboce o gráfico

Respostas:

a) D(f) = \mathbb{R} (definida para todos os reais)

b) Im(f) = [0, +\infty[:

  • Para x ≤ 1: x² varia de 0 a 1 (quando x=0 e x=1)
  • Para x > 1: 2x-1 > 1
  • O valor mínimo é 0 (em x=0)

c) Gráfico:

  • Parábola y=x² até x=1 (incluindo o ponto (1,1))
  • Reta y=2x-1 para x>1 (não inclui (1,1))
